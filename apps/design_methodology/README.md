## 문서 공통 기준

- 조합형 비교보다 단일 패턴 이해를 우선한다.
- 각 문서는 개념, 구조, 예시 시나리오, React + NestJS 연결점을 포함한다.
- 실제 구현 전에는 여기서 기준 패턴 하나를 먼저 선택한다.
