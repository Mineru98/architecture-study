# Integration Study Guide

`integration`은 시스템 간 데이터 전달과 메시징 전략을 단일 패턴 단위로 정리한 문서 모음이다.

## 데이터 전송 방식

- [RESTful](./restful/README.md)
- [GraphQL](./graphql/README.md)
- [RPC](./rpc/README.md)
- [WebSocket](./websocket/README.md)
- [MQTT](./mqtt/README.md)

## 메시징 및 통합 패턴

- [Pub/Sub](./pub-sub/README.md)
- [Message Queue](./message-queue/README.md)
- [Outbox Pattern](./outbox-pattern/README.md)

## 문서 공통 기준

- 프론트엔드와 백엔드의 연결 방식 자체를 하나씩 이해한다.
- 동기 호출과 비동기 메시징을 구분해서 본다.
- 학습 후 필요한 패턴을 조합해 실제 예제를 설계한다.
