# @vibe-architecture/react-headless

`@vibe-architecture/react-headless`는 clone한 seed-design `packages/react-headless`를 참고해 스터디에 필요한 기본 요소를 단일 패키지로 모아 둔 headless 레이어다.

현재 포함 요소:
- `Avatar`
- `Checkbox`
- `Collapsible`
- `Dialog`
- `Drawer`
- `Field`
- `FieldButton`
- `Fieldset`
- `Image`
- `Popover`
- `Portal`
- `Primitive`
- `Progress`
- `PullToRefresh`
- `RadioGroup`
- `Scrollable`
- `SegmentedControl`
- `Slider`
- `Snackbar`
- `Supports`
- `Switch`
- `Tabs`
- `TextField`
- `Toggle`
- `useControllableState`
- `Modal`

원본처럼 패키지별로 세분화하지 않고, 현재 스터디에서 빠르게 사용할 수 있게 한 workspace 패키지로 합쳤다.
