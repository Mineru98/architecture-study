# @vibe-architecture/react-headless

`@vibe-architecture/react-headless`는 clone한 seed-design `packages/react-headless`에서 필요한 패키지 아이디어만 모은 단일 패키지다.

반영한 최소 축:
- `Dialog`
- `Field`
- `Portal`
- `Primitive`
- `useControllableState`
- `Modal`

원본처럼 세분화된 다수 하위 패키지를 그대로 복제하지는 않고, 현재 스터디에 필요한 headless 로직만 단일 workspace 패키지에 모았다.
