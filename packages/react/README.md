# @vibe-architecture/react

`@vibe-architecture/react`는 clone한 seed-design `packages/react`의 방향을 따라 `components`, `primitive.ts`, `vars.ts`, `utils` 구조만 최소 반영한 패키지다.

현재 구조:
- `src/components`: 시각 컴포넌트
- `src/primitive.ts`: headless primitive re-export
- `src/vars.ts`: css token bridge
- `src/utils`: React 레이어 보조 타입

현재 범위는 학습용 최소 subset만 유지한다.
